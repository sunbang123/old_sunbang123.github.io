ESP32 LED 2개 Blink(점멸) 실습

한세대학교 컴퓨터공학과 22학번 김선영

브래드보드

브래드보드의 특징과 회로를 설계하는 방법
  1. 브래드보드: 저항, 커패시터, 다이오드 등의 다양한 부품을 수용할 수 있는 구멍이 있는 보드
  2. 프로토 타입 회로를 만들 때 쓰이며 회로를 빠르게 수정할 때 유용함.
  3. 브래드보드는 일반적으로 여러 가지 크기의 전자 부품을 수용할 수 있어서 많은 종류의 프로토타입을 만들 수 있음.
    4. (ex) LED 회로, 버튼 스위치 회로, 센서 회로, 아두이노 프로토타입
  4. 점퍼선: 브래드보드의 접촉점들을 연결하는데 사용되는 전선
  5. 브래드보드의 접촉점에 삽입되기 때문에 전기적인 연결이 완료되며 부품을 녹여 연결하는 솔더링이 필요하지 않음.

연결 방법
  1. 1-1. 브래드보드 위의 연결점 구조 파악
  2. - 행과 열의 구성: 브래드보드는 일렬로 배치된 행과 열의 구성으로 되어 있음.
  3. - 행과 열은 특정 규칙에 따라 연결되어 있어, 전자 부품을 브래드보드에 꽂으면 자동으로 연결되는 구조를 가지고 있음.
  4. 1-2. 전원 공급 연결: 전원 공급 라인에 USB를 연결하고, 전원과 그라운드를 이용해 회로를 구성함.
  5. 1-3. 전자 부품 연결: 전자 부품은 브래드보드의 연결점에 꽂음.
  6. 1-4. 회로 구성: 전원 공급 연결과 전자 부품 연결을 이용해 회로를 구성함.
  7. 1-5. 전원 공급 종료: 실험이 끝난 후에는 전원 공급을 종료함.

실습 : ESP32 LED 2개 Blink(점멸) 실험

회로도

코드

    int led1 = 22;  // LED 22 핀 번호
    int led2 = 23;  // LED 23 핀 번호
    
    void setup() {
        pinMode(led1, OUTPUT);
        pinMode(led2, OUTPUT);
    }
    
    void loop() {
        digitalWrite(led1, HIGH);
        digitalWrite(led2, LOW);
        delay(1000);
        digitalWrite(led1, LOW);
        digitalWrite(led2, HIGH);
        delay(1000);
    }

사용된 함수

- pinMode(): led1과 led2 핀을 출력 모드로 설정

- digitalWrite(): led1 핀을 HIGH(켜짐)으로 설정하고 led2 핀을 LOW(꺼짐)으로 설정

- delay(1000) : 1초 지연

실행 결과

저항 실험: 저항값이 다른 저항을 연결하면 LED 밝기에 차이가 있을까?

준비물
  1. - 저항1(주황색, 주황색, 갈색, 금색)
  2. - 저항2(빨간색, 보라색, 검은색, 검은색, 갈색)
  3. - 저항3(갈색, 빨간색, 갈색, 금색)

저항 계산법: 색상코드(4밴드, 5밴드, 6밴드)

  1. 4 자리 색상 코드는 첫 번째 띠와 두 번째 띠는 두 자리의 숫자를 나타내고, 세 번째 띠는 배율, 네 번째 띠는 오차를 나타내는 코드.
  2. 보통 앞뒤를 구별하기 위해, 세 번째 띠까지 간격이 좁고, 네 번째 띠는 간격이 넓게 표시 함.
  3. 계산법 예) 빨강, 빨강, 검정, 금색 : 22 x 1 = 22Ω, 오차 ±5%

  4. 저항1: 주황색, 주황색, 갈색, 금색 => 33 x 10 = 330Ω, 오차 ±5%
  5. 저항2: 빨간색, 보라색, 검은색, 검은색, 갈색 => 270 x 1 = 270Ω, 오차 ±270%
  6. 저항3: 갈색, 빨간색, 갈색, 금색 => 12 x 10 = 120Ω, 오차 ±5%

실험 결과

- 330Ω

- 270Ω

- 120Ω

  1. 옴의 법칙[I = V/R, <= V(전압), I(전류), R(저항) 전류는 저항에 비례하며 전압에 반비례]에 의해 저항이 클수록 전압은 낮아지고 전류는 조금 흐르게 됨.
  2. 반대로 저항이 작을수록 전압이 높아지고 전류는 많이 흐르게 되서 120Ω 저항을 연결한 LED의 밝기가 가장 센 것을 볼 수 있음
  3. LED 밝기 120Ω > 270Ω > 330Ω

실습을 통해 느낀점

- LED 2개 Blink 프로젝트를 진행하면서, 전기회로의 기본 원리와 옴의 법칙에 대해 배움.

- 특히, 저항이 클수록 전압은 낮아지고 전류는 적게 흐르고, 저항이 작을수록 전압은 높아지고 전류는 많이 흐르는 것을 확인할 수 있었음.

- 이를 통해 전기회로 설계에서는 적절한 저항 값을 선택하는 것이 중요하다는 것을 깨달았음.
