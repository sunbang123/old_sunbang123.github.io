ThingSpeak

한세대학교 컴퓨터공학과 22학번 김선영

ThingSpeak 란?

  1. ThingSpeak는 IoT 디바이스에서 수집한 센서 데이터를 클라우드 서버에 저장하고, 이를 분석하고 시각화하는 IoT 플랫폼
  2. 온라인 상에서 데이터 수집, 저장, 분석, 시각화, 공유 등의 IoT 서비스를 제공
  3. RESTful API를 제공하므로, 다른 응용 프로그램과 연동하여 데이터를 활용함

설정 방법

1. ThingSpeak 홈페이지에 회원가입을 함.

2. My Channels에서 New Channel을 만듦

3. New Channel에서 Field 이름을 지정함.

실습 : Arduino & ThingSpeak

1. Arduino IDE에서 ThingSpeak 라이브러리를 설치함.



2. 현재 연결된 WIFI 이름과 비밀번호를 확인한 다음 실습코드에 입력.

실습 코드

                                                
    #include <WiFi.h>
    #include <ThingSpeak.h>

    const char* ssid = "[WIFI 이름]";
    const char* password = "[WIFI password]";

    ...

3. ThingSpeak My Channel로 들어가서 Channel ID와 API키를 확인한 다음 실습코드에 입력.

실습 코드

                                                
    ...
    WiFiClient client;
    unsigned long ch1_ID = 2126281;
    const char* ch1_WriteKey = "AGP30QRISWWCJ0ZO";
    
    void setup() {
        Serial.begin(115200);
        delay(10);
        initWiFi();
        ThingSpeak.begin(client);
    }
    
    void loop() {
        int i;
        for (i = 0; i < 4; ++i) {
        ThingSpeak.writeField(ch1_ID, 1, i, ch1_WriteKey);
        delay(2000);
        }
    }
    
    void initWiFi() {
        WiFi.begin(ssid, password);
        while (WiFi.status() != WL_CONNECTED) {
        delay(500);
        Serial.println(".");
        }
    }
