ESP32 Blink(점멸) 실습

한세대학교 컴퓨터공학과 22학번 김선영

실습 보고서

지정보드 설정

1-1. 제어판>하드웨어 및 소리>장치관리자 창을 띄운다.

1-2. 제어판에서 포트 번호(COM)를 확인한 후 IDE에서 Tool>Port>(올바른 포트 번호) 를 선택한다.

1-3. IDE에서 Tools>Bord>ESP32>ESP32 DEV MODULE(장치 모듈)을 선택한다.

컴파일링 실행

2-1. SELECT 창에 ESP32 DEV MODULE를 선택한 후 -> 버튼을 눌러 컴파일(compile)해준다.

2-2. 컴파일링이 진행될동안 장치의 오른쪽 버튼(업로드 버튼)을 2초 누른다.

2-3. 컴파일이 성공하면 아래의 화면처럼 출력이 되는걸 볼 수 있습니다.

코드

    int LED_BUILTIN = 22; // 내장 LED가 연결된 디지털 핀 번호를 상수 22로 설정합니다.
    void setup() {
      pinMode(LED_BUILTIN, OUTPUT);
    }
    void loop() {
      digitalWrite(LED_BUILTIN, HIGH);
      delay(1000);                    
      digitalWrite(LED_BUILTIN, LOW); 
      delay(1000);                    
    }
사용된 함수

- setup() : 내장 LED가 연결된 디지털 핀을 출력으로 설정합니다.

- digitalWrite(LED_BUILTIN, HIGH/LOW) : 내장 LED를 켭니다.

- delay(1000) : 1초 지연.

에러

LED에는 극성이 있으며, 긴 다리가 양극, 짧은 다리가 음극입니다. 양극과 음극을 바르게 연결하지 않으면 제대로 작동하지 않습니다. 저항 옆에 양극을 연결해줍니다. 또한, LED 22 핀에 회로가 적절하게 연결되었는지 확인 합니다. (GPIO22, 저항, LED +)

아두이노 핀보드

설명

아두이노 핀보드는 아두이노 마이크로컨트롤러에서 사용되는 다양한 핀들을 관리하고 사용할 수 있게 해주는 보드입니다. 아두이노 핀보드는 대부분의 경우 직사각형 모양을 하며, 이 모양 안에 다양한 핀들이 위치하고 있습니다. GND는 아두이노 보드의 그라운드(Ground)를 의미합니다. 그라운드는 전기 회로에서 전하가 흐르는 기준점으로, 전기적으로 중립적인 지점입니다. 전기 회로를 구성할 때 GND 핀들은 전기 회로에서 전압을 조절하거나 전기 신호를 처리합니다.
