스마트디바이스 실습 개요

한세대학교 컴퓨터공학과 22학번 김선영

스마트 디바이스란?

마이크로프로세서
  1. 마이크로프로세서는 일반적으로 중앙처리장치(CPU), 기억장치, 연산장치 등을 포함하는 하나의 칩(chip)에 담겨있음.
  2. 이러한 칩은 CPU, 메모리, 인터페이스 등을 하나의 칩에 집적하는 기술을 통해 크기가 줄어들고 있음.
  3. 마이크로프로세서는 기술적 발전으로 인해 칩의 크기가 줄어들고 있음.
ARM
  1. ARM은 하드웨어 아키텍처(Instruction Set Architecture)를 기반으로 한 마이크로프로세서 제조업체.
  2. ARM 프로세서는 저전력, 고성능, 비용효율적인 장점으로 인해 스마트폰, 태블릿, IoT 장치 등에 널리 사용됨.

스마트폰
  1. 스마트폰 CPU는 ARM 마이크로프로세서 기술을 적용하여 매우 작고 성능이 뛰어남.
  2. 스마트폰의 CPU는 매우 작고 저전력이며, 하나의 칩에 CPU, 기억장치, 연산장치 등이 모두 집적되어 있음.
  3. 스마트폰은 과거에 비해 훨씬 더 가볍고, 작고, 성능이 우수해짐.
  4. 모바일 분야를 비롯한 다양한 분야에서 적용되어 새로운 기술과 서비스를 개발할 수 있게 됨.

스마트 디바이스
  1. 스마트 디바이스도 Embeded (computer) System 처럼 컴퓨터가 내장되어 있는 CPU.
  2. ESP32 : 저렴하고 고성능, 리소스가 많다. IDM만 있으면 변형해서 만들기 좋음.
  3. 종류 : 음향기기, 헬스케어, 영상기기, 구동기기(드론, 헬기) 웨어러블 기기, 스마트 가전, 디지털 사이니지 등...
  4. 특징 : 통신기술로 인터넷에 여녈되어 있음.
  5. 범위 : 스마트 센서, 부품, 엑추에이터, 앱세서리, 웨어러블 디바이스 Iot디바이스, 스마트센서 모듈 등...
IoT 구조의 요소
  1. 센서/엑츄에이터
  2. 제어부(마이크로프로세서)
  3. 스마트폰은 과거에 비해 훨씬 더 가볍고, 작고, 성능이 우수해짐.
  4. 통신부(인터넷, wifi, BT 등)

요약

- 스마트 디바이스는 마이크로프로세서 기술의 발전으로 인해 매우 작고 성능이 뛰어나며, 인터넷에 연결되어 다양한 서비스를 제공할 수 있는 기기.

- 스마트 디바이스는 음향기기, 헬스케어, 영상기기, 구동기기, 웨어러블 기기, 스마트 가전, 디지털 사이니지 등 다양한 분야에서 사용됨.

- 스마트 디바이스는 IoT 구조의 요소인 센서/엑츄에이터와 제어부(마이크로프로세서)로 이루어져 있음.

배운점

- 마이크로프로세서의 크기가 작아지면서 스마트 디바이스와 IoT 기술이 발전하고 우리 일상의 다양한 분야에서 사용되고 있다는 것을 알게 됨.

- 스마트 디바이스의 IoT구조를 배우면서 개념을 익히게 되서 좋았음.
