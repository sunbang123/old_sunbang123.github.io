아두이노 설정

아두이노 보드를 컴퓨터와 연결하여 사용하려면 해당 보드의 드라이버를 설치해야 합니다. 드라이버는 하드웨어와 컴퓨터 간의 통신을 가능하게 해주는 역할을 합니다. 아두이노 보드의 드라이버를 설치하면 컴퓨터는 해당 보드를 인식하고, 아두이노 IDE에서 보드를 사용할 수 있습니다. 따라서, 아두이노 보드를 사용하기 위해서는 드라이버 설치가 필수적입니다.

장치 드라이버 설치

CP210x USB 드라이버를 다운로드 해서 운영체제에 맞게 설치합니다.

지정보드 설정

1-1. 제어판>하드웨어 및 소리>장치관리자 창을 띄운다.

1-2. 제어판에서 포트 번호(COM)를 확인한 후 IDE에서 Tool>Port>(올바른 포트 번호) 를 선택한다.

1-3. IDE에서 Tools>Bord>ESP32>ESP32 DEV MODULE(장치 모듈)을 선택한다.

컴파일링 실행

코드

    int LED_BUILTIN = 2; // 리셋을 누르거나 보드에 전원을 공급할 때 설정 기능이 한 번 실행됩니다.
        void setup() { // 디지털 핀 LED 초기화_출력으로 내장됨.
        pinMode(LED_BUILTIN, OUTPUT);
    }
    
    // 루프문을 통해서 LED가 켜졌다 꺼졌다를 계속 반복하도록 합니다.
    void loop() {
        digitalWrite(LED_BUILTIN, HIGH);  // LED를 켭니다(HIGH는 전압 레벨)
        delay(1000);                      // 지연시간 1초
        digitalWrite(LED_BUILTIN, LOW);   // 전압을 낮게 하여 LED를 끕니다.
        delay(1000);                      // 지연시간 1초
    }

2-1. SELECT 창에 ESP32 DEV MODULE를 선택한 후 -> 버튼을 눌러 컴파일(compile)해준다.

2-2. 컴파일링이 진행될동안 장치의 오른쪽 버튼(업로드 버튼)을 2초 누른다.

2-3. 컴파일이 성공하면 아래의 화면처럼 출력이 되는걸 볼 수 있습니다.

에러

전력 공급이 안되거나 보드 설정이 잘못 되었을 경우 에러가 납니다.

위 에러는 ESP32 보드가 부트 모드로 실행되고 있어서 업로드에 실패한 것이다. 보드를 업로드 모드로 전환하면 문제를 해결할 수 있다.
업로드 버튼을 눌렀는지 확인하고 지정보드 설정이 바르게 되었는지 재확인합니다.
자세한 해결 방법: https://docs.espressif.com/projects/esptool/en/latest/troubleshooting.html
